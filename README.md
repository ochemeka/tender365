
# Tender365 - Minimal Scaffold
This is a minimal scaffold for Tender365 (Next.js + Prisma + NextAuth).

## Setup (local)
1. Copy `.env.example` to `.env` and fill values.
2. Install deps: `npm install`
3. Initialize Prisma: `npx prisma migrate dev --name init`
4. Seed admin: `npm run seed`
5. Run dev: `npm run dev`

## Notes
- This scaffold is a starting point. Fill in UI components and modules as needed.
